# Лабораторная 7: Авторизация и работа с заявками

## Студент: Пронин Вячеслав Константинович, ИУ5Ц-73Б

## Цель работы:
Завершение интерфейса пользователя в React с добавлением авторизации и работы с заявками через Redux Toolkit

## Задачи:
- Добавить авторизацию и регистрацию пользователей
- Реализовать работу с заявками (черновик, сохранение, отправка)
- Интегрировать Redux Toolkit с redux-thunk для управления состоянием
- Использовать кодогенерацию из Swagger для взаимодействия с API через Axios
- Реализовать личный кабинет пользователя

## Технологии:
- React + TypeScript
- Vite
- React-Bootstrap
- Redux Toolkit + redux-thunk
- Axios
- Swagger кодогенерация
- JWT / сессии

## Основные функции:
1. **Авторизация/регистрация** - вход, выход, сохранение сессии
2. **Управление заявками** - создание черновика, добавление услуг, сохранение, отправка
3. **Просмотр заявок** - таблица с фильтрацией по датам и статусам
4. **Личный кабинет** - смена пароля, данных пользователя
5. **Корзина услуг** - интерактивное управление черновой заявкой

## Порядок показа на защите:
1. Авторизация (логин: user, пароль: 1234)
2. Добавление и формирование заявки в режиме разработчика
3. Использование localStorage/cookie для демонстрации заявок в Postman
4. Пояснение кода: Redux Toolkit, Axios, кодогенерация из Swagger
5. Ответы на контрольные вопросы

## Контрольные вопросы:
- Схема Redux Toolkit (reducer, store, middleware)
- useContext
- Axios
- Local Storage

## Запуск:
```bash
npm install
npm run dev
```
Приложение доступно по адресу: http://localhost:3000/frontend

## Бэкенд:
Для работы требуется запущенный бэкенд из 4-й лабораторной (в Docker)
