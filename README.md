# Лабораторная 5: React SPA

## Студент: Пронин Вячеслав Константинович, ИУ5Ц-73Б

## Цель работы:
Разработка базового SPA на React с подключением к бэкенду через проксирование

## Задачи:
- Разработать три страницы фронтенд приложения на React и TypeScript
- Реализовать подключение к веб-сервису через fetch API
- Решить проблему CORS через проксирование Vite dev server
- Реализовать фильтрацию услуг с моками при недоступности бэкенда
- Использовать компоненты React-Bootstrap для UI

## Технологии:
- React + TypeScript
- Vite
- React-Bootstrap
- Fetch API
- Docker

## Страницы приложения:
1. **Главная страница** - статическое описание системы оценки облачного хостинга
2. **Страница списка тарифов** - карточки тарифов облачных провайдеров с фильтрацией
3. **Страница тарифа** - детальная информация о конкретном тарифе

## Особенности реализации:
- Навигационная панель navbar с breadcrumbs
- Проксирование запросов к бэкенду через Vite config
- Fallback на mock-данные при недоступности бэкенда
- Адаптивный дизайн на Bootstrap
