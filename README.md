# Лабораторная работа 6: Tauri desktop приложение

**Тема:** Оценка стоимости облачного хостинга для веб-проекта  
**Студент:** Пронин Вячеслав Константинович, ИУ5Ц-73Б  
**Ветка:** `Tauri`

#### Цель работы
Создание нативного desktop-приложения для просмотра тарифов облачного хостинга.

#### Задание
Создание простого нативного приложения на `Tauri` для интерфейса гостя (без авторизации и редактирования), состоящего из 3 страниц с фильтрацией и картинками. Подключить приложение к разработанному API через IP адрес в локальной сети (не `localhost`).

#### Особенности реализации
- Использование Tauri (Rust + WebView)
- Прямое подключение к бэкенду по IP локальной сети
- Сборка в .exe файл для Windows
- Адаптивный дизайн для desktop

#### Технологии
- Tauri
- React
- TypeScript
- Vite

#### Порядок показа
1. Запустить собранное Tauri приложение (.exe файл)
2. Показать работу с бэкендом через IP локальной сети
3. Продемонстрировать фильтрацию тарифов
4. Показать изображения из Minio
5. Сравнить IP сервера из консоли бэкенда и в коде приложения

#### Контрольные вопросы
- Виды нативных приложений
- Tauri архитектура
- Локальные сети и IP адресация
